# 灵触·随行 — 数据接口约定

## 算法

 `AlgorithmLayer.js` 输出一个**长度105**的一维数组。

```javascript
const grid = new Array(105);

// 编号顺序:
// grid[0-35]   前方盲文 F1-F6 (每模组6pin)
// grid[36-53]  左侧盲文 L1-L3
// grid[54-71]  右侧盲文 R1-R3
// grid[72-89]  后方盲文 B1-B3
// grid[90-104] DIY核心区 (3列×5行)

// 值:
//   0 = 无障碍
//   1 = 有障碍
//   2 = 高风险（仅DIY区grid[90-104]有效，盲文区的2会被自动降为1）
```

## 布局对应

```
         F1  F2  F3      ← grid[0-17]  前方上排
         F4  F5  F6      ← grid[18-35] 前方下排

    L1   DIY核心区   R1   ← grid[36-41] / grid[90-104] / grid[54-59]
    L2               R2   ← grid[42-47] / grid[60-65]
    L3               R3   ← grid[48-53] / grid[66-71]

         B1  B2  B3      ← grid[72-89] 后方
```

## 用户位置

`grid[100]` = 用户自身点（DIY区第4行第2列），固定为0。
